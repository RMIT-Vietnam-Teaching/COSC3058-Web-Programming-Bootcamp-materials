Setup

1️⃣ flex-direction

The flex-direction property determines the direction in which the flex items are placed within the container. This can drastically change the visual flow of the items.

"flex-direction: column": Flex items are arranged vertically, one on top of the other. This can be especially useful when designing vertical menus or lists.

1
2
3
4

"flex-direction: column-reverse": Similar to column, but the order of the items is reversed from that in the HTML. This is useful when you want the last element in your source code to appear at the top visually.

1
2
3
4

"flex-direction: row (default)": Flex items are placed side by side in a horizontal row. This is the default behavior when using display: flex, making it ideal for horizontal navigation bars or layouts.

1
2
3
4

"flex-direction: row-reverse": Flex items are arranged horizontally in the reverse order of the HTML source. This might be useful for right-to-left languages or specific design effects.

2️⃣ flex-wrap

In this section, we modify the container’s width (set to 300px) to demonstrate how flex items behave when there is limited space.

"flex-wrap: nowrap" (default):

  • By default, all flex items will try to fit on a single line.
  • If there isn’t enough space, items shrink and may eventually be clipped or overflow.
This setting is ideal when you want to ensure all items remain in a single row regardless of the container size.

3️⃣ justify-content

The justify-content property aligns flex items along the main axis. The effect varies when the main axis is horizontal versus vertical.

Note: The output of this property will vary if used with flex-direction: column since the main axis changes.

"justify-content: flex-start" (default):

  • Items are packed toward the start of the container (left in a row layout).
This is the default alignment and is useful for standard left-aligned layouts.

1
2
3
4

"justify-content: flex-end":

  • Items are aligned to the end of the container (right in a row layout).
This can be useful when you want to push content to the right.

1
2
3
4

"justify-content: center":

  • Flex items are centered along the main axis.
This is ideal for centering elements within the container for a balanced look.

1
2
3
4

"justify-content: space-around":

  • Items are distributed with equal space around them.
This provides equal spacing on both sides of each item, which can help in creating a visually even layout.

1
2
3
4

"justify-content: space-between":

  • Items are distributed so that the first item is at the start and the last is at the end, with equal space between the items.
This is useful for layouts where you need maximum space utilization between elements.

4️⃣ align-items

The align-items property aligns flex items along the cross-axis (perpendicular to the main axis). Adding a specific container height allows you to clearly see how items align vertically when in row layout.

"align-items: flex-start": Items are aligned to the top of the container.

1

3
4

"align-items: flex-end": Items are aligned to the bottom of the container.

"align-items: center": Items are centered along the cross-axis, resulting in a balanced vertical layout.

"align-items: stretch": When items do not have a set height, they will stretch to fill the container's height. This can help maintain consistency in element sizes across the layout.

5️⃣ align-content

The align-content property aligns rows of flex items when there is extra space in the cross-axis. This property only affects multi-line flex containers (i.e., when flex-wrap is set to wrap or wrap-reverse).

"align-content: stretch (default)":

  • Rows stretch to fill the container, creating an even distribution.
  • This effect is visible only when there are multiple lines of flex items.
