CSS Grid Showcase (Detailed Explanations)

This page demonstrates important CSS Grid features in ascending order of complexity. Each section contains a code snippet, a live demo, and a detailed explanation to help you understand how and why each property or technique is used.

1. Basic Grid Container

By setting display: grid on a container, all direct children become grid items. This is the foundation of CSS Grid. You can later define columns, rows, and other properties to control how these items are laid out.

Key Insight: Even with no columns or rows explicitly defined, the container will automatically place items in the first row, then create new rows as needed. The gap property (also written as grid-gap in older specs) adds space between grid cells.

/* HTML */ <div class="grid-container"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div> /* CSS */ .grid-container { display: grid; gap: 10px; }

2. Basic 3-Column Grid

Using grid-template-columns: 1fr 1fr 1fr creates three equal-width columns. The 1fr unit represents a fraction of the remaining space, making it easy to create flexible layouts that adjust to the container’s width.

Key Insight: The fr unit is unique to CSS Grid and is ideal for fluid layouts. You can combine it with fixed values (like 200px) or percentages to create more complex sizing rules.

/* HTML */ <div class="three-column"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> <div>Item 4</div> <div>Item 5</div> <div>Item 6</div> </div> /* CSS */ .three-column { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 10px; }

3. Grid Columns and Rows

In this example, we explicitly define both columns and rows. The first row and second row are each set to 100px in height, giving you more direct control over the layout. Grid items will automatically populate these cells in row-major order.

Key Insight: Combining explicit row definitions with explicit columns helps you create a more structured layout. For instance, you might use a fixed-height header row or a sidebar column.

/* HTML */ <div class="grid-columns-rows"> <div>A</div> <div>B</div> <div>C</div> <div>D</div> </div> /* CSS */ .grid-columns-rows { display: grid; grid-template-columns: 1fr 1fr; grid-template-rows: 100px 100px; gap: 5px; }

4. Gaps in Grid

The gap property creates consistent spacing between rows and columns. It is especially helpful when you want to avoid using margins on each grid item. You can also set row-gap and column-gap individually.

Key Insight: Before the gap shorthand was introduced, we used grid-row-gap and grid-column-gap separately. Now, gap (or grid-gap) is the recommended approach for all modern browsers.

/* HTML */ <div class="grid-gap"> <div>1</div> <div>2</div> <div>3</div> <div>4</div> </div> /* CSS */ .grid-gap { display: grid; grid-template-columns: 1fr 1fr; gap: 20px; }

5. Placing Grid Items

You can manually position items within the grid using grid-column and grid-row. Here, the first item spans two columns by using grid-column: span 2. This explicit placement allows you to create magazine-style layouts or highlight certain elements.

Key Insight: You can also specify exact line numbers for more precise control, such as grid-column: 1 / 3 or grid-row: 2 / 4, where “1 / 3” means from line 1 (inclusive) to line 3 (exclusive).

/* HTML */ <div class="grid-placement"> <div class="item1">Item 1 (span 2 columns)</div> <div class="item2">Item 2</div> <div class="item3">Item 3</div> </div> /* CSS */ .grid-placement { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; } .item1 { grid-column: span 2; }

6. Implicit Rows

If you have more items than the defined rows, Grid will create additional rows automatically. The grid-auto-rows property specifies the height for these implicitly created rows. In this demo, each new row is 60px high.

Key Insight: You can also use auto, min-content, max-content, or fit-content() for implicit rows, which is useful if your items vary in size.

/* HTML */ <div class="implicit-rows"> <div>Item 1</div> <div>Item 2 (taller content)</div> <div>Item 3</div> <div>Item 4</div> <div>Item 5</div> <div>Item 6</div> <div>Item 7</div> </div> /* CSS */ .implicit-rows { display: grid; grid-template-columns: repeat(3, 1fr); grid-auto-rows: 60px; gap: 10px; }

7. Aligning Grid Items

justify-items and align-items control the alignment of items within their cells. In this demo, we use center for both properties to place each box in the middle of its cell. We also add Flexbox to each grid cell to ensure text is perfectly centered horizontally and vertically.

Key Insight: For more complex alignment, you can target individual items with justify-self or align-self. These properties override justify-items and align-items for a specific item.

/* HTML */ <div class="alignment-demo"> <div>Box 1</div> <div>Box 2</div> <div>Box 3</div> <div>Box 4</div> <div>Box 5</div> <div>Box 6</div> </div> /* CSS */ .alignment-demo { display: grid; grid-template-columns: repeat(3, 100px); grid-template-rows: repeat(2, 100px); gap: 10px; justify-items: center; align-items: center; } .alignment-demo > div { background: #b2dfdb; display: flex; justify-content: center; align-items: center; width: 100%; height: 100%; }

8. Grid Template Areas

grid-template-areas lets you name different parts of the layout and arrange them in a grid visually. Each item has a grid-area name (e.g., header, sidebar, etc.). This is especially useful for building page templates (header, main content, sidebar, footer) that are easy to rearrange later.

Key Insight: Named grid areas are a powerful way to keep your CSS more readable and maintainable. Instead of memorizing line numbers, you can rearrange sections by editing the area strings in your CSS.

/* HTML */ <div class="grid-areas"> <div class="header">Header</div> <div class="sidebar">Sidebar</div> <div class="content">Content</div> <div class="footer">Footer</div> </div> /* CSS */ .grid-areas { display: grid; grid-template-columns: 1fr 2fr; grid-template-rows: auto auto auto; grid-template-areas: "header header" "sidebar content" "footer footer"; gap: 10px; }

9. Grid Auto Flow

grid-auto-flow determines how items are placed if the grid is already partially filled. The dense keyword allows the browser to fill empty cells when possible, which is helpful for masonry-like layouts. Notice how items spanning multiple columns or rows don’t necessarily leave gaps behind.

Key Insight: grid-auto-flow: dense can sometimes reorder your items visually, which might not be ideal for accessibility or reading order. Use it carefully if you rely on source order for content flow.

/* HTML */ <div class="auto-flow"> <div class="wide">Wide Item</div> <div class="tall">Tall Item</div> <div class="normal">Normal</div> <div class="normal">Normal</div> <div class="normal">Normal</div> <div class="normal">Normal</div> </div> /* CSS */ .auto-flow { display: grid; grid-template-columns: repeat(3, 1fr); grid-template-rows: repeat(3, 80px); gap: 10px; grid-auto-flow: dense; } .wide { grid-column: span 2; } .tall { grid-row: span 2; }

10. Responsive Grid (Media Queries)

By combining CSS Grid with media queries, you can adapt the layout for different screen sizes. In this demo, the grid has four columns on screens wider than 600px, but collapses into a single column for narrower screens.

Key Insight: This approach is straightforward but requires manually defining breakpoints for each layout change. It’s useful when you have specific design requirements for different device widths.

/* HTML */ <div class="responsive-grid"> <div>Box 1</div> <div>Box 2</div> <div>Box 3</div> <div>Box 4</div> </div> /* CSS */ .responsive-grid { display: grid; grid-template-columns: repeat(4, 1fr); gap: 10px; } @media (max-width: 600px) { .responsive-grid { grid-template-columns: 1fr; } }

11. Responsive Grid (auto-fit/minmax)

This method uses repeat(auto-fit, minmax(200px, 1fr)) to create a truly responsive grid without manually writing media queries. Each column is at least 200px wide but can expand to fill remaining space. Columns automatically wrap to a new line as needed.

Key Insight: Using auto-fit or auto-fill with minmax() is a powerful way to create fluid, responsive grids. It’s often a simpler approach than multiple media queries for many layouts.

/* HTML */ <div class="auto-fit-grid"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> <div>Item 4</div> <div>Item 5</div> </div> /* CSS */ .auto-fit-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 20px; }
