Beginner CSS Grid Layout Examples with Extended Advanced Layouts

This guide starts with basic grid layouts and gradually introduces more advanced techniques. Explore each example and read the insights to learn how to build and tweak responsive and complex grid layouts.

Example 1: Your First Grid - A Simple 2×2 Grid

What's happening: A fixed-size 2×2 grid layout where each cell has defined dimensions.

  • display: grid sets the element as a grid container.
  • grid-template-columns: 150px 150px and grid-template-rows: 100px 100px define fixed sizes.
  • Items fill each cell following the document order.

Use this when: You need a rigid structure with predictable sizing.

.container { display: grid; grid-template-columns: 150px 150px; grid-template-rows: 100px 100px; }
<div class="container"> <div class="grid-item">1</div> <div class="grid-item">2</div> <div class="grid-item">3</div> <div class="grid-item">4</div> </div>

This simple layout places items in a 2×2 fixed grid.

Example 2: Adding Gap Between Grid Items

What's happening: The gap property adds uniform spacing between cells.

  • gap: 20px creates even space between both rows and columns.
  • Unlike margins, the gap only appears between items—not around the outer edge of the grid.

Use this when: You want a cleaner layout without manually setting margins.

.container { display: grid; grid-template-columns: 150px 150px; grid-template-rows: 100px 100px; gap: 20px; }
<div class="container"> <div class="grid-item">1</div> <div class="grid-item">2</div> <div class="grid-item">3</div> <div class="grid-item">4</div> </div>

Example 5: Using the repeat() Function

What's happening: The repeat() function offers a shorthand for defining repetitive tracks.

Use this when: You need many equally sized columns or rows without redundant code.

Example 6: Two Unequal Columns (1/3 and 2/3)

What's happening: Here, a 1:2 ratio is used so one column takes 1/3 while the other gets 2/3 of the space.

  • The layout dynamically resizes while preserving the ratio.

Use this when: You need emphasis on one column such as a sidebar or a main article.

.container { display: grid; grid-template-columns: 1fr 2fr; grid-template-rows: 100px; gap: 10px; }

Example 7: Perfectly Centered Element

What's happening: The place-self property centers an item both vertically and horizontally.

  • place-self: center combines justify-self and align-self in one rule.
  • This method works regardless of element dimensions.

Use this when: You want to quickly center an element in its grid cell.

.container { display: grid; grid-template-columns: 1fr; grid-template-rows: 1fr; height: 200px; } .centered { place-self: center; width: 150px; height: 100px; }
<div class="container"> <div class="centered grid-item">Centered Element</div> </div>
Centered Element

Effortlessly centers an element within its parent container.

Example 9: Simple Sidebar Layout

What's happening: A layout with a fixed sidebar paired with a flexible main content area is created. The header and footer span the full width.

Example 12: Placing Grid Items

What's happening: This demo shows how to explicitly position grid items. Notice how the first item spans two columns.

Example 13: Implicit Rows

What's happening: When there are more items than defined rows, Grid automatically creates extra rows. Here, each implicit row is set to 60px high using grid-auto-rows.

  • grid-auto-rows: 60px ensures consistent height for all automatically generated rows.
  • You can also use values like min-content or max-content for dynamic sizing.

Use this when: Your layout contains a variable number of items and you want consistent row heights.

.container { display: grid; grid-template-columns: repeat(3, 1fr); grid-auto-rows: 60px; gap: 10px; }
<div class="container"> <div>Item 1</div> <div>Item 2 (taller)</div> <div>Item 3</div> <div>Item 4</div> <div>Item 5</div> <div>Item 6</div> <div>Item 7</div> </div>

Example 14: Aligning Grid Items

What's happening: This demo shows how to use justify-items and align-items to center grid items in their cells.

  • justify-items: center horizontally centers items in each grid cell.
  • align-items: center vertically centers items.
  • You can override these using justify-self or align-self for specific items.

Use this when: You need a consistent alignment across grid items or want to control individual item positioning.

.container { display: grid; grid-template-columns: repeat(3, 100px); grid-template-rows: repeat(2, 100px); gap: 10px; justify-items: center; align-items: center; }

Example 15: Grid Template Areas

What's happening: With grid-template-areas, you can assign names to sections of your grid. This method makes rearranging layout regions easier.

  • grid-template-areas defines a visual map of your layout.
  • Each child is assigned to an area using the grid-area property.

Use this when: You need a semantic and easily maintainable page layout, such as for headers, sidebars, and footers.

.container { display: grid; grid-template-columns: 1fr 2fr; grid-template-rows: auto auto auto; grid-template-areas: "header header" "sidebar content" "footer footer"; gap: 10px; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .content { grid-area: content; } .footer { grid-area: footer; }

  • grid-template-areas places each “card” into specific cells, allowing some to overlap horizontally or vertically.

Example 17: Grid Auto Flow

What's happening: The grid-auto-flow property defines how items fill in the grid when not explicitly placed. The dense keyword tries to fill gaps for a tighter layout.

  • grid-auto-flow: dense can reorder items to eliminate gaps.
  • This is useful for masonry-style layouts, but be cautious as it can affect source order.

Use this when: You want a compact layout that maximizes the use of available space.

.container { display: grid; grid-template-columns: repeat(3, 1fr); grid-template-rows: repeat(3, 80px); gap: 10px; grid-auto-flow: dense; } .wide { grid-column: span 2; } .tall { grid-row: span 2; }

Example 18: Responsive Grid (Media Queries)

What's happening: A responsive grid is created with media queries. On wider screens it shows four columns; on screens smaller than 600px, it collapses to one column.

  • Breakpoints allow for targeted layout adjustments based on screen size.
  • This approach is useful when you have specific design requirements for different devices.

Use this when: You want to manually control how your grid adapts across device sizes.

.container { display: grid; grid-template-columns: repeat(4, 1fr); gap: 10px; } @media (max-width: 600px) { .container { grid-template-columns: 1fr; } }

Example 19: Responsive Grid (auto-fit/minmax)

What's happening: This demo uses repeat(auto-fit, minmax(200px, 1fr)) to automatically adjust the number of columns based on available space.

  • minmax(200px, 1fr) sets a minimum column width of 200px and allows it to grow.
  • The grid automatically wraps items as the container size changes, eliminating the need for media queries.

Use this when: You need a fluid and responsive layout without manually managing breakpoints.

.container { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 20px; }
