Exercise 7: Reordering with Order Property

Goal: Create a flex container with three boxes and use the order property so that Box 2 appears first.

Hint: Assign lower order values to the box you want to appear first.

The target layout:

Box 1
Box 2
Box 3

Your current layout:

Box 1
Box 2
Box 3
Show Solution 
/* CSS for Exercise 7 */
.container {
  display: flex;
}
.container .box:nth-child(1) {
  order: 2;
}
.container .box:nth-child(2) {
  order: 1;
}
.container .box:nth-child(3) {
  order: 3;
}

/* Explanation:
   - The 'order' property determines the visual order of the boxes.
   - Here, Box 2 (the second child) is given the lowest order (1) so that it appears first.
*/

Exercise 8: Responsive Flexbox Layout

Goal: Build a responsive flex container that displays boxes in a row on larger screens and stacks them vertically on screens 600px wide or less.

Hint: Use a media query to change flex-direction at the breakpoint.

The target layout:

Box 1
Box 2
Box 3

Your current layout:

Box 1
Box 2
Box 3
Show Solution 
/* CSS for Exercise 8 */
.container {
  display: flex;
  flex-direction: row;
}
@media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}

/* Explanation:
   - By default, the container displays its boxes in a row.
   - The media query changes the flex-direction to column on screens 600px wide or less,
     causing the boxes to stack vertically.
*/
